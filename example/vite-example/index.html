Using with Proxy Server: This example uses a local proxy server to avoid CORS issues.

To run this example:
  1. Install dependencies: npm install
  2. Start both servers: npm start
  3. Proxy server will run on port 3010

If you see connection errors, make sure the proxy server is running.
